-----------------------------------------------
   Melody training basic kit - ver 1.1
-----------------------------------------------

メロディーの作成補助ソフトです。
１６小節にコードを割当て、メロディーを
作成していきます。ベースラインとして、
練習の伴奏としても使えます。


[起動方法]
フォルダ内の exeファイル をダブルクリックします。


[動作環境(確認したもの)]
64 bit版  Windows10 

32 bit版では、たぶん動作しません

[起動に必要なソフト]
.NET Framework 4.8 のバージョン

（バージョン 4 以上なら互換性があるそうです）

（ダウンロードサイト）
https://docs.microsoft.com/ja-jp/dotnet/framework/install/on-windows-10

---------------------------
         操作方法
---------------------------

1) コードを選択して、[make] ボタンを押すと
　 構成音からメロディーを作成します。

   ※ コードはメジャー、マイナーの 7th コードです。

   チェックボックスにチェックのある部分のみ変更されます。

　 {b-♯}　の選択で、符号の表示は変化します。

   音符は << 英文字 + (-,b,#) >> の２文字で表現されています。

   c d e f g a b C D E F G A  の2オクターブ弱の範囲が再生できます。

   １小節は4分音符4個の構成になっていますので、
　 半角 8文字分の記入となります。
   誤記入がありますと、再生やデータのセーブもうまくいきません。

　 b と # は混在可です。例として d#ebd#eb だと ミ♭が４回となります。

　コードの選択はメロディ発行時以外は使いませんので、
　コードの下の欄の構成音は自由に変更可能です。


2) [play]ボタンで再生、[stop]ボタンで停止します。

   {Loop}, {Tempo},{Pattern} でリピート回数とテンポ、進行パターンが設定できます。


3) [Save]ボタンでデータファイルを保存できます。
　 ファイル名はテキストボックス欄に ~.dat で記入します。

   exe ファイルと同じフォルダにデータファイルは保存されます。

4) [Open]ボタンでデータファイルを読込みます。
